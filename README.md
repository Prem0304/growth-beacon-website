# Growth Beacon - Digital Marketing Agency Website

Welcome to the official codebase for the **Growth Beacon** website. Growth Beacon is a premium digital marketing agency specializing in Enterprise SEO, Paid Acquisition, CRO, Brand Intelligence, CRM Automation, Full-Funnel Analytics, Social Media Marketing, and High-Authority Content.

## 🚀 Technologies Used
* **Frontend:** Clean HTML5, Vanilla CSS3 (Custom styling, glassmorphism, responsive grid systems, and smooth animations).
* **Interactions:** Custom modern Vanilla JS for interactive cards, counts animation, mobile navbar toggle, and smooth scrolling.
* **Hosting (Local):** Python HTTP Server.

## 📁 File Structure
```
growth-beacon-website/
├── index.html        # Main webpage layout, sections, and forms
├── style.css         # Design tokens, variables, typography, and responsive styles
├── script.js        # Dynamic UI scripts and counter animation
├── assets/           # Media files & brand assets
│   ├── growth_beacon_logo.jpg  # Option 16 (AI Sparkle Chart) master logo
│   └── growth_beacon_logo.svg  # Vector logo design
└── README.md         # Repository documentation
```

## 🛠️ Local Development & Testing

To run the project locally, run a local server in the project root:

```bash
# Using Python
python -m http.server 8000 --bind 127.0.0.1
```

Once running, navigate to `http://127.0.0.1:8000` in your web browser.
